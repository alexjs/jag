# 1985 Jaguar XJS 5.3 HE — Singapore

Living documentation for my 1985 Jaguar XJS 5.3 HE (V12), registered and run in
Singapore. This repo collects specs, maintenance history, parts references, and
notes that may be useful to other XJS owners — particularly anyone keeping one
of these alive in a tropical climate under Singapore's regulatory regime
(COE, LTA inspections, classic/vintage scheme).

## The car

| | |
|---|---|
| Model | Jaguar XJS 5.3 HE |
| Year | 1985 |
| Engine | 5.3 L V12, High Efficiency (HE / Fireball head) |
| Transmission | GM TH400 3-speed automatic |
| Location | Singapore |

## A note on what's public

This is a public repo. Identifying and personal details — VIN, registration
plate, insurance and ownership documents, invoices with names/addresses —
live in `private/`, which is gitignored. If you're contributing or copying
this structure: keep it that way.
