# Reference documents

Manuals, wiring diagrams, and reference material. Only freely redistributable
material lives here; everything else is linked.

## In this repo

- [`manual/`](manual/) — the official **Jaguar XJ-S HE Repair Operation
  Manual** (scanned, 388 pp) with a navigation index. Hosted here with rights
  confirmed directly with JLR (out of copyright; confirmed 2026).

## External resources

- **Kirby Palm, *Experience in a Book — Help for the Jaguar XJ-S Owner*** —
  the single best XJS resource; ~740 pages of owner experience. Free download
  from the author via [jag-lovers.org](http://www.jag-lovers.org/xj-s/book/)
  (copyrighted by Palm, so linked rather than hosted here).
- **[Jaguar sounds library](http://jaguar.professional.org/sounds/index.php)** —
  recordings of Jaguar engines and mechanical noises; occasionally handy for
  diagnosis by ear.
- **Jag-Lovers forums** — the [XJ-S forum](https://forums.jag-lovers.com/) and
  V12 engine archives.
- **AJ6 Engineering** — technical notes on V12 cooling and tuning.
