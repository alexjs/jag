# XJ-S Repair Operation Manual + H.E. Supplement — navigation index

[`xj-s-he-repair-manual.pdf`](xj-s-he-repair-manual.pdf) (388 PDF pages) is
actually **two documents bound together**:

1. **PDF pages 1–313** — the base Jaguar XJ-S Repair Operation Manual
   (covers the original pre-HE car)
2. **PDF pages 314–388** — the **XJ-S H.E. Supplement to the Repair Operation
   Manual** — the part that describes a 1981-on 5.3 HE. Where the supplement
   and the base manual disagree, *the supplement wins* for this car.

The scan has no bookmarks, so this index maps sections to PDF pages. The
manual's own per-operation contents listing is on **PDF pages 3–17** and is
fully legible. A searchable OCR text layer has been added to the PDF —
useful for Cmd-F, but on a scan this rough always verify numbers against the
page image.

Manual page references look like `12–3` (section 12, page 3); operation
numbers like `12.13.26` — the leading pair is the section number.
