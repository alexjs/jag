# Specifications — 1985 XJS 5.3 HE

Starter reference. Values marked *(verify)* are commonly cited for the HE but
should be checked against the factory manual for this exact car before relying
on them.

## Engine

| | |
|---|---|
| Configuration | 60° V12, SOHC per bank, 5343 cc |
| Head | HE "Fireball" (Michael May design) high-swirl combustion |
| Compression | 12.5:1 (Euro spec) *(verify which market this car is)* |
| Injection | Lucas digital EFI |
| Ignition | Lucas CEI (amplifier mounted in the engine V — heat-sensitive) |

## Driveline

| | |
|---|---|
| Transmission | GM Turbo-Hydramatic 400, 3-speed automatic |
| Final drive | Salisbury/Dana, Powr-Lok LSD on most HEs *(verify ratio — typically 2.88:1)* |
